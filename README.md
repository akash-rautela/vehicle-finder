## ✨ About the Project

**Vehicle Finder** is a smart web application that helps users discover the best **2-wheelers 🏍️** and **4-wheelers 🚗** based on:

- 💰 Budget  
- ⚙️ Preferences  
- 🎯 Usage needs  

No more confusion — just **personalized vehicle recommendations**.

---

## 🚀 Features

- 🔍 Smart recommendation system  
- 💸 Budget-based filtering  
- 🏍️ Bike & 🚗 car suggestions  
- ⚡ Fast and responsive UI  
- 🎨 Clean and modern design  
- 📱 Mobile-friendly

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/akash-rautela/vehicle-finder.git
cd vehicle-finder

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🧠 How it Works

1. User enters preferences (budget, type, usage)
2. System processes inputs
3. Best matching vehicles are recommended
